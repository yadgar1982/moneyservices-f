# Money Services Management System (Frontend)

## Overview

The Money Services Management System is a full-stack MERN application designed for businesses involved in money transfer, currency exchange, and financial transaction management.

This repository contains the **React.js frontend** that provides an intuitive, responsive, and secure user interface for administrators, branch operators, and authorized staff.

---

## Features

### Authentication & Security

* Secure user login
* JWT-based authentication
* Role-based authorization
* Protected routes
* Session management

### User & Staff Management

* User registration
* User profile management
* Role-based access control
* Branch assignment
* Account holder management

### Branch Management

* Create, update, and manage branches
* Branch-specific operations
* Branch-wise reporting

### Currency Management

* Multi-currency support
* Currency configuration
* Exchange rate management

### Transactions

* Credit transactions
* Debit transactions
* Money transfers
* Currency exchange
* Transaction history
* Transaction search and filtering

### Reports

* Customer statements
* Branch-wise reports
* Currency-wise reports
* Transaction reports
* Balance reports
* Financial summaries
* Exportable reports

### Additional Features

* Webcam integration
* Digital signature canvas
* Printing support
* Data export functionality
* Email confirmation workflows
* Responsive design

## Installation

```bash
git clone <repository-url>

cd frontend

npm install

npm run dev
```

---

## Environment Variables

Create a `.env` file and configure the required frontend environment variables.

Example:

```env
VITE_API_URL=http://localhost:5000
```
